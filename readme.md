# Technologies used
* Python
* Flask
* Bootstrap
* Sqlite

# About the project
* This is a fullstack project which topic is a Employee Management System
* This is my fullstack project using a different stack. Backend is enhanced with flask and its tools, which I found very useful and convenient to manage; and the Frontend powered with bootstrap.
* Like any of my projects I do, this is a full dedicated project with the following features:
  * Login, signup, logout
  * Add, delete, update your company, your employees, and the departments of your selected companies
  * Secured credentials in the database
  * A search bar that retrieves according the query (Could be a employee, company, or department)
  * Many more stuff ...

# How to get started
* Clone it
* In the root, run: python app.py
* Initialize the db
  * flask db init	                    
  * flask db migrate -m "commit with a message"	    
  * flask db upgrade
